# SmartBudgetApp

## Technology Stack

This application is built using the following technologies:

- **Java Spring Boot**: A powerful framework for building web applications and RESTful services.
- **PostgreSQL**: A robust, open-source relational database.
- **Maven**: A build automation tool for Java projects.

## Overview

SmartBudgetApp is designed to help users manage their finances more effectively using a modern technology stack that supports scalability and efficiency. The application provides features for tracking expenses, creating budgets, and generating reports.

## Features

- User authentication and authorization.
- Expense tracking with categorization.
- Budget creation and monitoring.
- Reporting capabilities to visualize spending habits.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/swanco-byte/SmartBudgetApp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SmartBudgetApp
   ```
3. Install the dependencies:
   ```bash
   mvn install
   ```
4. Run the application:
   ```bash
   mvn spring-boot:run
   ```
5. Access the application at `http://localhost:8080`.
